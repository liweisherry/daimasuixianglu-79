
## 150. Evaluate Reverse Polish Notation
- 题目链接：https://leetcode.com/problems/evaluate-reverse-polish-notation/description/
- 文章讲解/视频讲解： 

### 💭 First Thought
- 用stack来储存每个数字和运算符号

### ✅ Final Approach

```
class Solution(object):
    op_map = {
        '+': add,
        '-': sub,
        '*': mul,
        '/': lambda a, b: int(a / b)  # truncate toward zero
    }

    def evalRPN(self, tokens: List[str]) -> int:
        stack = []
        for token in tokens:
            if token not in self.op_map:
                stack.append(int(token))
            else:
                op2 = stack.pop()
                op1 = stack.pop()
                stack.append(self.op_map[token](op1, op2))
        return stack.pop()
```

### ⚠️ Mistakes / Gotchas
- 使用functions来运行计算
### Complexity
Time: O(n)

## 239. Sliding Window Maximum
- 题目链接：https://leetcode.com/problems/sliding-window-maximum/description/
- 文章讲解/视频讲解： 

### 💭 First Thought
- 需要一个队列，这个队列呢，放进去窗口里的元素，然后随着窗口的移动，队列也一进一出，每次移动之后，队列告诉我们里面的最大值是什么。
- 其实队列没有必要维护窗口里的所有元素，只需要维护有可能成为窗口里最大值的元素就可以了，同时保证队列里的元素数值是由大到小的。

### ✅ Final Approach

```
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        result = []
        dq = deque()
        
        for i in range(len(nums)):
            # 1. 踢出左侧过期索引
            if dq and dq[0] < i - k + 1:
                dq.popleft()
                
            # 2. 核心灵魂：保持单调递减（内卷淘汰）
            # 新元素 nums[i] 进场，把队尾所有比它小的元素全部踢掉！
            while dq and nums[dq[-1]] < nums[i]:
                dq.pop()  # 注意：是从右侧 pop！
                
            # 3. 新人索引正式入队
            dq.append(i)
            
            # 4. 窗口成型，队首 dq[0] 永远是当前窗口最大值的索引
            if i >= k - 1:
                result.append(nums[dq[0]])
                
        return result
```

### ⚠️ Mistakes / Gotchas
- 使用functions来运行计算
### Complexity
Time: O(n)
